# KToolsCore — Справочник и архитектура

## Проект
Модульный WoW-аддон для Legion 7.3.5.
Репозиторий: github.com/kawfia/KToolsCore

**Ядро (KTools)** — общий UI, навигация, безопасный вызов модулей, общие библиотеки.
**Модули** — регистрируются в ядре при загрузке, рендерят свои настройки внутри общего контейнера.

Каждый модуль имеет **собственную версию** и **собственный SavedVariables**.

---

## Архитектура

### Ядро не знает о модулях
Модули сами регистрируются при загрузке. Ядро предоставляет:

```lua
KTools:RegisterModule(name, module)   -- вызывается из модуля
KTools:SafeCall(module, method, ...)  -- ядро вызывает методы модулей
```

Критическая ошибка в одном модуле не роняет остальные.

### Хардкод запрещён
Никаких жёстко зашитых значений кроме `.toc` файлов: имя разработчика, версия, название.
Все остальные значения — из `.toc` или SavedVariables.

### Локализация
Каждый аддон содержит свою локаль:
```
locale/
  load.xml    enUS.lua    ruRU.lua
```

### Структура файлов
`.toc` → `load.xml` → `.lua` файлы.
`load.xml` — только в подпапках, описывает файлы нижележащего уровня.

### Комментарии в файлах
Каждый `.lua` файл начинается с комментария — полный путь к файлу, начиная с папки аддона:
```lua
-- KTools/core/ui.lua
```

### .toc файлы
- `## Notes:` — только английский текст
- `## Notes-ruRU:` — русский перевод отдельной строкой

## Файловая структура

```
KTools/                          — ядро
  KTools.toc                     — точка входа аддона
  init.lua                       — инициализация, регистрация Ace
  core/
    load.xml                     — список core-файлов
    minimap.lua                  — кнопка миникарты (LibDBIcon)
    settings.lua                 — SavedVariables ядра
    ui.lua                       — главное окно, навигация, RegisterModule, SafeCall
  lib/
    load.xml                     — подключение библиотек
  locale/
    load.xml                     — список файлов локали
    enUS.lua                     — английская локаль
    ruRU.lua                     — русская локаль
  media/
    minimapButton_dx5.blp        — иконка кнопки миникарты

KToolsAutoloot/                  — модуль авто-лута
  KToolsAutoLoot.toc             — точка входа

KToolsTPH/                       — модуль трекера фарма
  KToolsTPH.toc                  — точка входа

.claude/
  CLAUDE.md                      — инструкции для Claude
  MEMORIES.md                    — контекст предыдущих сессий
  REFERENCE.md                   — этот файл
  rules/
    red-lines.md                 — абсолютные запреты
    architecture.md              — перенесено в REFERENCE.md
  tasks/
    KTools.md                    — документация ядра
    KToolsAutoLoot.md            — документация модуля авто-лута
    KToolsTPH.md                 — документация модуля TPH
  reference/
    KToolsAutoLoot.html          — диаграмма интерфейса авто-лута
    core.h                       — enum ItemClass, ItemBonding, InventoryType (Legion)
    lib/                         — библиотеки (не коммитятся, копировать вручную)
    addons/                      — референсные аддоны
  hooks/
    sync.sh                      — синхронизация репо (SessionStart + UserPromptSubmit)
    session-start.sh             — вызывает sync.sh
    user-prompt-submit.sh        — вызывает sync.sh
    pre-tool-use.py              — блокирует rm и опасные git-операции
    stop-remind.sh               — напоминание обновить MEMORIES.md
  patch/                         — патчи (способ 3 генерации кода)
```

## Ограничения

- Целевой API: **Legion 7.3.5**. Retail/Classic/ERA функции недопустимы.
- Перед использованием WoW API функции — убедиться что она существует в Legion 7.3.5.
- Публичный API (функции KTools, события, ключи SavedVariables) — менять только после обсуждения.
- Подробнее: `.claude/rules/red-lines.md`
