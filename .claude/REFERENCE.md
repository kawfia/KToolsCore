# KToolsCore — Справочник проекта

## Общее описание
Модульный WoW-аддон для Legion 7.3.5.
Репозиторий: github.com/kawfia/KToolsCore

**Ядро (KTools)** предоставляет общую оболочку: кнопку миникарты, главное окно с навигацией между модулями, безопасный вызов модулей и общие библиотеки.
**Модули** регистрируются в ядре при загрузке и рендерят свои настройки внутри общего контейнера.

---

## Структура проекта

```
KTools/              — ядро
KToolsAutoloot/      — модуль авто-лута
KToolsTPH/           — модуль трекера фарма (на обсуждении)
lib/                 — общие библиотеки (подключаются ядром -> переносятся в KTools/lib по востребованию)
.claude/             — рабочая папка Claude Code
```

Каждый модуль имеет **собственную версию** и **собственный SavedVariables**.

---

## Связь ядра и модулей

Ядро предоставляет функцию регистрации. Модуль вызывает её при инициализации:

```lua
KTools:RegisterModule(name, module)
```

Вызовы методов модулей происходят через защищённую обёртку ядра —
критическая ошибка в одном модуле не роняет остальные.

Ядро предоставляет единый UI-контейнер — модули рендерят свои настройки
внутри него, не создавая собственных окон.

## Важные ограничения

- Целевое API: **Legion 7.3.5**. Retail/Classic функции недопустимы.
- Изменение публичного API (функции в namespace KTools, события, ключи SavedVariables) — только после обсуждения.
- Все изменения — через `.patch` файлы. Прямой коммит в main запрещён.
- Подробнее: `.claude/rules/red-lines.md`
