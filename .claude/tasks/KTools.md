## KTools — Ядро

**Модульное ядро аддона**

## Назначение
Предоставляет три компонента всем модулям:
1. **UI-оболочка** — кнопка миникарты + главное окно с навигацией между модулями
2. **Безопасный вызов** — ошибка в модуле не роняет ядро и остальные модули
3. **Общие библиотеки** — поставляет зависимости для всех модулей

## Команда
`/ktools` — открывает главное окно
`/ktl` - алиас для `/ktools`

## Опции (ESC → Интерфейс → Модификации)
> Сркыть/показать + отдельно заблокировать кнопку миникарты.

## Интерфейс
Главное окно содержит навигацию между зарегистрированными модулями.
Каждый модуль рендерит свою панель внутри общего контейнера.

## Описание кнопок интерфейса
> Уточнить после проверки репозитория

## Файловая структура аддона
```
KTools/KTools.toc          — точка входа, зависимости аддона
KTools/init.lua            — инициализация, AceAddon
KTools/core/load.xml       — список core-файлов
KTools/core/minimap.lua    — кнопка миникарты (LibDBIcon)
KTools/core/settings.lua   — SavedVariables ядра
KTools/core/ui.lua         — главное окно, навигация, RegisterModule, безопасный вызов
KTools/lib/load.xml        — подключение библиотек
KTools/locale/load.xml     — подключение файлов локали
KTools/locale/enUS.lua     — английская локаль
KTools/locale/ruRU.lua     — русская локаль
KTools/media/minimapButton_dx5.blp  — иконка кнопки миникарты
```

## SavedVariables
> Уточнить — минимально: состояние кнопки миникарты (показать/скрыть)
