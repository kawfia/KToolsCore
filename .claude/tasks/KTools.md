# KTools — Ядро

## Назначение
Модульное ядро аддона. Решает три задачи:
1. **UI-оболочка** — общий интерфейс с навигацией между модулями, кнопка у миникарты.
2. **Безопасный вызов модулей** — если модуль падает с критической ошибкой, остальные продолжают работать.
3. **Общие библиотеки** — поставляет зависимости для модулей, снижая их индивидуальный вес.

## Публичный API
> Заполнить после проверки репозитория

### Регистрация модуля
```lua
-- Модуль регистрирует себя в ядре
KTools:RegisterModule(name, module)
```

### Безопасный вызов
```lua
-- Ядро вызывает методы модулей через защищённую обёртку
KTools:SafeCall(module, method, ...)
```
