# KToolsAutoLoot — Авто-лут

## Описание
Модуль для KTools. Автоматически подбирает лут по настраиваемым правилам.
Команда: `/ktloot`

Настройки разбиты на два места:
- **Главное окно** — основные правила лута (открывается через `/ktloot` или навигацию KTools)
- **ESC → Интерфейс → Модификации** — вспомогательные опции поведения

---

## Интерфейс

Диаграмма: `.claude/reference/KToolsAutoLoot.html`

Главное окно содержит **две вкладки**: «Быстрые настройки» и «Пользовательский список».

### Вкладка: Быстрые настройки

#### Быстрые категории

| Кнопка | Описание | Статус |
|---|---|---|
| Золото | Подбирать монеты с лута | ✓ |
| Реагенты | Расходуемые материалы для крафта | ✓ |
| Рецепты | Все рецепты, включая уже изученные | ✓ |
| Маунты | Предметы-маунты | ✓ |
| Питомцы | Боевые питомцы | — |
| Квестовые предметы | Предметы с типом Quest | — |
| Валюта | Жетоны и валюты (ресурсы оплота, мана и т.п.) | ✓ |
| Сила артефакта | Предметы для прокачки артефактного оружия | ✓ |

#### Фильтр по качеству и ilvl

Для каждого качества: чекбокс + поле `ilvl >=` + фильтры привязки.

> **Правило ilvl:** ограничение действует **только на выбранное качество**.

| Качество | Описание поля ilvl | NoB | BoE | BoP | Статус |
|---|---|---|---|---|---|
| Poor | Минимальный ilvl для серых | ✓ | ✓ | ✓ | ✓ |
| Common | Минимальный ilvl для белых | ✓ | ✓ | ✓ | ✓ |
| Uncommon | Минимальный ilvl для зелёных | ✓ | ✓ | ✓ | ✓ |
| Rare | Минимальный ilvl для синих | ✓ | ✓ | ✓ | ✓ |
| Epic | Минимальный ilvl для фиолетовых | ✓ | ✓ | ✓ | ✓ |
| Legendary | Минимальный ilvl для оранжевых | ✓ | ✓ | ✓ | ✓ |

NoB — не привязан, BoE — при экипировке, BoP — при подборе.

---

### Вкладка: Пользовательский список

Список конкретных itemID. Работает независимо от категорий и фильтра качества.

| Колонка | Описание | Статус |
|---|---|---|
| ✓ (checkbox) | Включить / выключить эту запись | ✓ |
| Тип (Автозаполняется по ID) | Категория для отображения: Эквип / Валюта / Рецепты / Реагенты | ✗ |
| ID | Поле ввода itemID | ✓ |
| Название предмета | Автозаполняется по ID | ✓ |
| ilvl >= | Минимальный ilvl (опционально) | частично |
| Удалить | Удалить эту запись из списка | ✗ |

## Скрытые настройки (ESC → Интерфейс → Модификации)

| Опция | Описание | Статус |
|---|---|---|
| Без подтверждения BoP | Автоматически подтверждать диалог привязки при подборе BoP | — |
| Закрыть лут при скиннинге | Закрывать окно лута при начале каста снятия шкур; после завершения каста — открыть окно шкуры | — |
| Мгновенное закрытие пустого лута | Закрывать окно лута сразу если оно пустое | — |
