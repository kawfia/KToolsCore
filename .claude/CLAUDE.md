## Генерация кода

### Способ 1 — полный файл
Новый файл **или** больше двух правок на файл:
- вывести файл целиком в блоке кода
- первой строкой — путь к файлу (начиная с папки аддона)

### Способ 2 — замена фрагмента
До 5 строк, не более двух правок на файл:
- первый блок — старый код
- второй блок — новый код
- указать путь к файлу над каждым блоком

---

## После генерации кода

**Слепой ревью** — вызвать Agent без описания задачи:
- содержимое сгенерированного кода
- `.claude/rules/red-lines.md`
- `.claude/rules/architecture.md`
- `.claude/tasks/<модуль>.md` — если код касается модуля
- `.claude/REFERENCE.md`

Ожидаемый ответ агента: `OK` или список нарушений.
