## Lua 5.1
**Аддон для World of Warcraft Legion (7.3.5)**

- каждый файл является реализацией только своей зоной ответственности.
- общая структура проекта
```
.toc
init.lua
/locale/load.xml
/
```

### load.xml
- находится только в подпапках
- на него ссылается только .toc файл
- из него ссылки на все компоненты в текущей либо нижних папках.

### locale
- дирректория для поддержки локали текущего аддона всех языков.
