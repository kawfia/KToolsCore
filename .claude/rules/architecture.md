# Architecture — KTools

Статус: **пре-альфа**. Архитектурные решения принимаются сейчас и фиксируются здесь.
Этот файл — история решений, не спецификация. Обновляется по мере принятия решений.

---

## Что такое KTools (цель)

Набор инструментов для WoW-игрока. Не один монолит, а модульная система где каждый модуль решает конкретную задачу и может работать независимо.

---

## Принятые архитектурные решения

### Решение 1 — Мульти-версионная стратегия
**Статус:** обсуждается, не финализировано.

Варианты:
- A) Один аддон, runtime-определение версии через `WOW_PROJECT_ID`
- B) Отдельные `.toc` файлы, общий Lua-код, compat-обёртки
- C) Форк репо на каждую версию

**Текущий выбор:** вариант B — отдельные `.toc` + `Compat/` папка.
Причина: позволяет поддерживать разные версии из одного репо без дублирования кода.

> Если это решение меняется — обновить этот файл и lua-conventions.md.

---

### Решение 2 — Модульная система
**Статус:** принято.

Каждый модуль = отдельный `.lua` файл, доступный через `KTools.ModuleName`.
Модули не знают друг о друге напрямую — только через основной namespace.
Порядок загрузки контролируется через `.xml`.

---

### Решение 3 — Нет LibStub/LibBabble/AceX по умолчанию
**Статус:** принято (пре-альфа).

Причина: в пре-альфа стадии лишние зависимости усложняют. Когда появится конкретная задача где нужна библиотека — обсудить и зафиксировать здесь.

## Структура модулей (планируется)

```
KTools/
├── Core/
│   ├── Init.lua        ← namespace, feature flags
│   ├── Events.lua      ← централизованная шина событий
│   └── Settings.lua    ← SavedVariables, defaults
├── Compat/
│   ├── Retail.lua      ← Retail-specific API
│   ├── Classic.lua     ← Classic Era API
│   └── Cata.lua        ← Cata-specific API
└── Modules/
    └── (будущие модули)
```

---

## Что НЕ принято решать пока

- Какие конкретно инструменты будут в KTools (обсуждается)
- UI-фреймворк (custom frames vs LibWindow vs etc.)
- Система настроек (custom vs AceConfig)

Эти решения принимаются когда появляется конкретная задача, требующая их.
