# KTools: Auto Loot
**Модуль для KTools**
**Автоматически подбирает лут по настраиваемым правилам.**

---

## Возможности

### Быстрые категории
Одним чекбоксом включить сбор:
- **Золото** — голда и серебро с лута
- **Валюта** — жетоны и валюта (реусрсы оплота, мана, etc.)
- **Квестовые предметы**
- **Реагенты** — расходуемые материалы для крафта
- **Рецепты** — собирает все рецепты, даже которые изучены
- **Сила артефакта** — итемы для усиления артефактного оружия
- **Маунты**
- **Питомцы**
- **Токены** — Legion лут-мешки и токены (Сундуки требующие навыка вскрытия, Добыча кровавого охотника и т.п.)

### Фильтр по качеству и ilvl
Подбирать предметы начиная с выбранного качества (Плохое — Легендарное).
Опционально: минимальный ilvl для точного совпадения качества.

**Если опция активна - ограничение ilvl действует только на выбранное качество. Любые предметы выше качеством подбираются без ограничений!**

### Кастомный список предметов
Список конкретных itemID с опциональным минимальным ilvl.
Поддерживает любой предмет независимо от категории.

### Профили
- Создание, удаление,импорт экспорт

### Дополнительно
- **Без подтверждения BoP** — автоматически подтверждает привязку при подборе
- **Закрыть лут при снятии шкур** — закрывает окно лута когда начинается каст скиннинга; после завершения каста открывается окно со шкурой
- Пустое окно лута закрывается мгновенно.
